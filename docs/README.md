# Documentación del Proyecto

Esta carpeta contiene documentación y recursos de apoyo para el proyecto MoneyTracker.

## Estructura

### `/deployment`
Documentación relacionada con el despliegue del proyecto:
- `DEPLOY.md` - Guía detallada de despliegue en Vercel
- `CHECKLIST-DEPLOY.md` - Checklist paso a paso para el despliegue

### `/design-proposals`
Propuestas de diseño UI/UX creadas durante el desarrollo:
- `design-proposals.html` - Propuestas iniciales de diseño general
- `form-proposals.html` - Propuestas para el formulario de gastos
- `form-proposals-creative.html` - Propuestas creativas adicionales para el formulario

**Nota**: Estos archivos son solo de referencia histórica. El diseño final implementado está en el código del frontend.
